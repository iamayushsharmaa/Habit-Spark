# 🧩 Habit Tracker App
A  Habit Tracker app built with Kotlin for Android and Firebase for backend. This app helps users build and maintain good habits by tracking their daily progress, offering visual insights, and syncing data securely with a firestore.

## 📝 Overview
The Habit Tracker App enables users to create, manage, and monitor their habits over time. With intuitive UI, offline support, and firebase integration, it’s designed for users who want to stay consistent and accountable on their self-improvement journey.

## 🚀 Key Features
✅ Create and Track Habits – Add new habits, mark them daily, and see streaks.

📊 Progress Visualization – View daily, weekly, and monthly stats to stay motivated.

📆 Calendar View – Easily track completed habits on an interactive calendar.

📴 Offline Support – Work seamlessly even without internet; syncs when online.

👤 User Profiles – Authenticate users and show personalized habit history.
