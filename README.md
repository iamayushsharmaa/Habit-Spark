# 🧩 Habit Tracker App
A  Habit Tracker app built with Kotlin for Android and Ktor for the backend. This app helps users build and maintain good habits by tracking their daily progress, offering visual insights, and syncing data securely with a cloud backend.

## 📝 Overview
The Habit Tracker App enables users to create, manage, and monitor their habits over time. With intuitive UI, offline support, and backend integration, it’s designed for users who want to stay consistent and accountable on their self-improvement journey.

## 🚀 Key Features
✅ Create and Track Habits – Add new habits, mark them daily, and see streaks.

📊 Progress Visualization – View daily, weekly, and monthly stats to stay motivated.

☁️ Cloud Sync – Save and retrieve habits securely with backend integration.

📆 Calendar View – Easily track completed habits on an interactive calendar.

📴 Offline Support – Work seamlessly even without internet; syncs when online.

👤 User Profiles – Authenticate users and show personalized habit history.

## 🛠 Tech Stack
📱 Frontend (Android)
Kotlin

Jetpack Compose – Modern, declarative UI toolkit

Room Database – Local storage

Firebase Auth – User authentication

Coroutines & Flow – Asynchronous programming

Hilt / Koin – Dependency injection

## 🌐 Backend (API)
Ktor (Kotlin)

MongoDB – NoSQL database for storing habits and user data

JWT Authentication – Secure API endpoints

Ktor Client – For HTTP communication between app and server
